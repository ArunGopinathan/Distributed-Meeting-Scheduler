This is a Eclipse Dynamic Web Project
Add this to your eclipse workspace
make sure you add this to Apache Tomcat Web Server

Run:
http://localhost:8080/DistributedMeetingSchedulerWebService/DMSWebService/Authenticate/Arun/1234

you can use any values in Arun or 1234 . As of now it does not allow special characters i guess

It returns the response

<User><FirstName>Arun</FirstName><LastName>Gopinathan</LastName><MavEmail>a</MavEmail><AndroidDeviceId /></User>
